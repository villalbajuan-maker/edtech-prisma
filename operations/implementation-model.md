# Implementation Model

PRISMA instruments should be designed for institutional self-application.

The core value is that once an institution activates an instrument and students complete it, insights are immediately available.

## Default Flow

```text
Institution selects instrument
-> creates application/cohort
-> invites students
-> students complete instrument
-> scoring runs automatically
-> reports and exports become available
-> institution acts on recommendations
```

## Optional Support

Services may be offered as add-ons:

- setup;
- configuration;
- interpretation session;
- implementation planning;
- follow-up review;
- custom reporting.

## Principle

PRISMA should not depend on consulting delivery to create core value.

Consulting may amplify value, but the instrument must stand on its own.
