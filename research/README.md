# Research

This directory stores evidence, interview notes, source tracking, and institutional learning.

## Research Rules

- Separate assumptions from evidence.
- Record source, date, and context.
- Capture buyer language verbatim when available.
- Track economic pain.
- Track current alternatives.
- Track moments of institutional urgency.
