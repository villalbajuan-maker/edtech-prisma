# Scoring Model

## Scoring Layers

PRISMA Permanencia should use three scoring layers.

## 1. Dimension Score

Each dimension receives an independent score and level.

Suggested levels:

- Strength
- Adequate
- Attention
- Priority

## 2. Priority Of Accompaniment

The student receives an institutional priority level.

Suggested levels:

- Low priority
- Moderate priority
- High priority
- Immediate priority

This is not a dropout prediction label.

It is a support prioritization signal.

## 3. Dominant Profile

The instrument assigns a dominant profile based on risk pattern, not only average score.

Initial profiles:

- Prepared and connected
- Academically vulnerable
- Vocationally uncertain
- Financially fragile
- Overloaded
- Institutionally disconnected
- Integrated alert

## Combination Rules

The model must identify risky combinations, for example:

- low vocational clarity + low belonging;
- financial pressure + need to work;
- low self-regulation + low academic preparation;
- weak support network + high stress;
- unrealistic expectations + low help-seeking confidence.
