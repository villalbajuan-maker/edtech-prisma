# Modelo De Implementación

Los instrumentos PRISMA deben diseñarse para autoaplicación institucional.

El valor central es que, una vez la institución activa el instrumento y los estudiantes lo completan, los insights quedan disponibles inmediatamente.

## Flujo Base

```text
La institución selecciona el instrumento
-> crea aplicación o cohorte
-> invita estudiantes
-> los estudiantes completan el instrumento
-> corre la puntuación automática
-> se habilitan reportes y exportaciones
-> la institución actúa sobre recomendaciones
```

## Apoyo Opcional

Los servicios pueden ofrecerse como complementos:

- configuración;
- parametrización;
- sesión de interpretación;
- planeación de implementación;
- revisión de seguimiento;
- reportes personalizados.

## Principio

PRISMA no debe depender de entrega consultiva para crear valor central.

La consultoría puede amplificar el valor, pero el instrumento debe sostenerse por sí mismo.
