# Bitácora De Decisiones

Este archivo registra decisiones estratégicas para que el proyecto no se diluya.

## 2026-04-18

### Decisión: PRISMA es una línea separada de negocio y producto

PRISMA no se mezcla con el repositorio ni con la operación actual de EduTech Marketing.

### Decisión: El modelo de negocio se centra en instrumentos

El activo económico central es el instrumento diagnóstico, no la hora consultiva.

Pueden existir servicios complementarios, pero el negocio principal es diseñar, preparar, estructurar, implementar y comercializar instrumentos diagnósticos digitales de alta calidad.

### Decisión: Primer mercado: educación superior

El primer segmento objetivo son universidades e instituciones de educación superior en Colombia.

### Decisión: Primer instrumento: PRISMA Permanencia

El primer instrumento se enfoca en preparación, fragilidad temprana, riesgo de permanencia y rutas de acompañamiento para estudiantes de ingreso.
