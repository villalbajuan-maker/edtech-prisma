# Modelo De Puntuación

## Capas De Puntuación

PRISMA Permanencia debe usar tres capas de puntuación.

## 1. Puntuación Por Dimensión

Cada dimensión recibe una puntuación y un nivel independiente.

Niveles sugeridos:

- Fortaleza
- Adecuado
- Atención
- Prioridad

## 2. Prioridad De Acompañamiento

El estudiante recibe un nivel de prioridad institucional.

Niveles sugeridos:

- Prioridad baja
- Prioridad moderada
- Prioridad alta
- Prioridad inmediata

Esta no es una etiqueta de predicción de deserción.

Es una señal para priorizar acompañamiento.

## 3. Perfil Dominante

El instrumento asigna un perfil dominante según el patrón de fragilidad, no solo por promedio.

Perfiles iniciales:

- Preparado y conectado
- Académicamente vulnerable
- Vocacionalmente incierto
- Financieramente frágil
- Sobrecargado
- Institucionalmente desconectado
- Alerta integral

## Reglas De Combinación

El modelo debe identificar combinaciones de riesgo, por ejemplo:

- baja claridad vocacional + baja pertenencia;
- presión financiera + necesidad de trabajar;
- baja autorregulación + baja preparación académica;
- red de apoyo débil + alto estrés;
- expectativas irreales + baja confianza para pedir ayuda.
