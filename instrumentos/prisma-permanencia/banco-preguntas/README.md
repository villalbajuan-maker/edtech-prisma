# Banco De Preguntas

Este directorio contendrá el banco de preguntas de PRISMA Permanencia.

## Propósito

Diseñar ítems que permitan detectar condiciones de preparación, fragilidad temprana y necesidades de acompañamiento sin reducir el instrumento a una encuesta superficial.

## Reglas Iniciales

- Cada pregunta debe mapear a una dimensión o subdimensión.
- Cada pregunta debe tener una razón diagnóstica clara.
- Evitar lenguaje punitivo, clínico o estigmatizante.
- Diseñar para interpretación institucional, no solo para respuesta individual.
- Mantener trazabilidad entre pregunta, puntuación, perfil y recomendación.
