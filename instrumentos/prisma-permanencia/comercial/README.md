# Comercialización De PRISMA Permanencia

Este directorio contendrá la oferta comercial, narrativa de venta, modelos de licencia y materiales para compradores institucionales.

## Compradores Principales

- rectoría;
- vicerrectoría académica;
- dirección de permanencia;
- dirección de bienestar;
- dirección de planeación o calidad;
- decanaturas;
- direcciones de programa.

## Tesis Comercial

PRISMA Permanencia no vende preguntas.

Vende una forma estructurada de convertir el ingreso estudiantil en una estrategia temprana de permanencia.
