# Implementación Institucional

Este directorio documentará cómo una institución activa, aplica y utiliza PRISMA Permanencia.

## Flujo De Implementación

```text
Configurar cohorte
-> activar instrumento
-> invitar estudiantes
-> recolectar respuestas
-> generar puntuaciones
-> habilitar reportes
-> activar rutas de acompañamiento
```

## Principio

La institución debe poder obtener valor sin depender de una consultoría extensa.

El apoyo externo puede mejorar la adopción, pero el instrumento debe estar diseñado para sostenerse como activo digital.
