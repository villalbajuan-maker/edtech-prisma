# Hoja De Ruta PRISMA

## Fase 0: Fundamento Estratégico

Objetivo: definir la categoría, el primer mercado, el primer campo de dolor y el primer instrumento.

Estado: en curso.

Entregables:

- tesis de categoría;
- hipótesis de problema;
- análisis forense de microproblemas;
- hipótesis de solución;
- brief del primer instrumento;
- mapa de portafolio;
- opciones de modelo de negocio.

## Fase 1: Metodología De PRISMA Permanencia

Objetivo: convertir el ajuste problema-solución en una arquitectura completa de instrumento.

Entregables:

- dimensiones y subdimensiones finales;
- estrategia de ítems;
- banco de preguntas V1;
- reglas de puntuación;
- perfiles de salida;
- matriz de recomendaciones;
- guía de lenguaje ético.

## Fase 2: Reportes Y Salidas Institucionales

Objetivo: definir exactamente qué recibe una institución inmediatamente después de aplicar el instrumento.

Entregables:

- especificación del reporte para estudiante, si aplica;
- especificación del reporte ejecutivo;
- especificación del reporte operativo;
- requerimientos de tablero por cohorte;
- esquema de exportación;
- flujo de decisión institucional.

## Fase 3: Empaque Comercial

Objetivo: convertir el instrumento en una oferta vendible.

Entregables:

- modelo de licencia;
- hipótesis de precio;
- narrativa comercial;
- mapa de compradores;
- paquete piloto;
- manejo de objeciones;
- plantilla de propuesta.

## Fase 4: Requerimientos De Plataforma

Objetivo: definir la tecnología mínima necesaria para operar la biblioteca de instrumentos.

Entregables:

- modelo de datos;
- roles y permisos;
- requerimientos del motor de puntuación;
- requerimientos del generador de reportes;
- flujo institucional;
- versionamiento de instrumentos;
- modelo de privacidad y uso de datos.

## Fase 5: Prototipo Tecnológico Viable

Objetivo: construir solo después de que el instrumento y el modelo comercial estén suficientemente claros.

Posibles entregables:

- aplicación de instrumento;
- motor de puntuación;
- generador de reportes;
- tablero institucional;
- exportaciones;
- catálogo básico.
