# Solution Hypothesis

## Core Hypothesis

If a higher education institution can apply a digital diagnostic instrument at student entry and immediately transform responses into preparation profiles, risk types, accompaniment priorities, and operational recommendations, then it can intervene earlier, allocate permanence resources more precisely, and create a baseline for student permanence management.

## Solution Mechanism

```text
Diagnose early
-> segment precisely
-> assign support routes
-> prioritize resources
-> improve permanence management
```

## Required Capabilities

The first instrument must include:

- early entry diagnosis;
- multidimensional scoring;
- priority of accompaniment;
- dominant profile;
- combination rules;
- recommendations by profile;
- aggregate institutional reporting;
- operational prioritized list;
- ethical language;
- follow-up route.

## Value Hypothesis

Institutions will adopt PRISMA Permanencia if it helps them answer:

- Who needs accompaniment first?
- What type of support does each student need?
- Which risks are present before academic failure appears?
- Which programs or cohorts have greater fragility?
- How should support teams prioritize limited capacity?
- How can permanence strategy move from reaction to prevention?
