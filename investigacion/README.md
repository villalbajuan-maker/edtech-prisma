# Investigación

Este directorio almacena evidencia, notas de entrevistas, seguimiento de fuentes y aprendizaje institucional.

## Reglas De Investigación

- Separar supuestos de evidencia.
- Registrar fuente, fecha y contexto.
- Capturar lenguaje real de compradores cuando exista.
- Rastrear dolor económico.
- Rastrear alternativas actuales.
- Rastrear momentos de urgencia institucional.

## Líneas Iniciales De Investigación

- permanencia y deserción;
- operación institucional en educación superior;
- éxito estudiantil y acompañamiento;
- mapa de compradores y presupuestos;
- alternativas competidoras;
- contexto colombiano de educación superior.
